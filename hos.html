Harvey's of Stretton

This is a web application that I created in my computing HND course as my first thing created with HTML and CSS. This was a very simple database manager that allowed the user to view, edit and delete information from different databases within the fictional company the app was created for.

Tools and Technologies used

  • HTML
  • CSS
  • JavaScript
  • PHP
  • MySQL
  • HiediSQL
  • XAMMP

Project purpose and goal

A fictional company needed a web application that would allow them to access their company databases, view and modify the information within them and be able to search for specific customers or vehicles.

Problems and solutions

As it was done for an assignment for my HND course, there were some limitations to what could be done. First off, we had to use applications that aren't necessarily what would be used in the real world (i.e., HiediSQL and XAMMP) to create the databases to be linked using PHP.

The main problem though was learning the basics as this was the first front end development project I did, so I was learning HTML, CSS and PHP whilst creating the application.

Results

As a first front end development project I was happy with what I created however I think now after having 1 year of knowledge I think I could recreate it in a much better way. I could make the UI the web application uses much better mainly because it wasn't responsive, and the PHP could be rewritten to make it more readable.
